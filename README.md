# Task 6: Create a Strong Password and Evaluate Its Strength

## Objective
To understand what makes a password strong and test it using an online password strength checker.

## Tool Used
https://passwordmeter.com

---

## Password Testing Results

### 1️⃣ Weak Password
Password: password123  
Score: 43%  
Complexity: Good  

Observation:
- No uppercase letters
- No special symbols
- Sequential numbers (123)
- Easy to guess

---

### 2️⃣ Medium Password
Password: Password123  
Score: 75%  
Complexity: Strong  

Observation:
- Added uppercase letter
- Contains numbers
- No special symbols
- Better than previous password

---

### 3️⃣ Strong Password
Password: P@ssw0rd!2026#  
Score: 100%  
Complexity: Very Strong  

Observation:
- Contains uppercase and lowercase letters
- Contains numbers
- Contains special symbols
- Long and complex
- No simple patterns

---

## Key Learnings

- Password length is very important.
- Use uppercase, lowercase, numbers, and symbols.
- Avoid sequential numbers and dictionary words.
- Avoid simple patterns.
- Longer passwords are harder to crack.

---

## Common Password Attacks

1. Brute Force Attack – Trying all possible combinations.
2. Dictionary Attack – Using common word lists to guess passwords.

---

## Conclusion

Strong passwords significantly improve security and reduce the risk of hacking. 
Using complex characters and longer length makes passwords harder to break.
